RetroBSD jest portem 2.11 BSD targetowanym na rodzinę mikrokontrolerów PIC32. Mikrochipy te są wyposażone w 128 kB pamięci RAM i 512 bajrów pamięci flash.

Wśród głównych cech systemu:

  • niewielkie wymagania,</li>
  • ochrona pamięci (MMU),</li>
  • system wielozadaniowy,</li>
  • API zgodne ze standardami POSIX,</li>
  • możliwość posiadania kompilatora i natywne budowanie aplikacji lub rekompilowanie całego systemu.</li>
    </ul>
